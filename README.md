# crud table

> crud table learn vue.js

## Installation

```bash
# clone this repo
$ git clone https://github.com/achtarudin/crud-table.git nameYourFolder

# enter the folder
$ cd nameYourFolder

# install dependencies
$ npm install

# serve with hot reload at localhost:8080
$ npm run dev

# build for production with minification
$ npm run build
```
